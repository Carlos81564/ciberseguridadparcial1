Unidad 1: Ataques de Ciberseguridad

Principales Tipos de Ataques

Ataque Descripción Ejemplos Prevención
Phishing Engaño para obtener credenciales o datos sensibles mediante correos falsos. Correos de bancos falsos, enlaces maliciosos. Filtros antispam, formación de usuarios, verificación de remitentes.
Malware Software malicioso que daña o controla sistemas. Virus, ransomware, troyanos, spyware. Antivirus actualizado, no ejecutar archivos desconocidos.
DDoS Saturación de un servicio con tráfico falso para dejarlo inaccesible. Ataques a sitios web, servidores de juegos. CDN, firewalls, mitigación de tráfico.
SQL Injection Inyección de código SQL en formularios para manipular bases de datos. Robo de datos de usuarios, borrado de tablas. Consultas preparadas, validación de entrada.
XSS (Cross-Site Scripting) Inyección de scripts en páginas web que se ejecutan en el navegador. Robo de cookies, redirección maliciosa. Escapado de salida, CSP, validación de entrada.
Man-in-the-Middle (MITM) Intercepción de comunicaciones entre dos partes. Wi-Fi público, falsos certificados SSL. HTTPS, VPN, verificación de certificados.
Brute Force Intento masivo de contraseñas hasta acertar. Ataques a cuentas de correo, SSH. Límites de intentos, 2FA, contraseñas fuertes.
Zero-Day Explotación de vulnerabilidades desconocidas por el fabricante. Ataques avanzados a sistemas críticos. Actualizaciones rápidas, monitoreo de amenazas.
Spoofing Falsificación de identidad (IP, email, DNS). IP spoofing, email spoofing. DKIM, SPF, verificación de origen.
Ransomware Cifrado de archivos exigiendo rescate. WannaCry, LockBit. Backups offline, no pagar rescate, EDR.

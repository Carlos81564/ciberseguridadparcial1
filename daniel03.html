Unidad 1: Ataques de Ciberseguridad

Tipos de Ataques

Ataque Descripción Impacto Prevención / Mitigación Imagen
Phishing Correos electrónicos o mensajes que suplantan identidades legítimas. Robo de credenciales, suplantación. Verificar remitentes, no clicar links, 2FA.
Ransomware Cifra archivos y pide rescate. Pérdida de datos, interrupción. Backups, actualizaciones, antivirus.
Malware Programas maliciosos para dañar sistemas o robar info. Robo de información, ralentización. Antivirus, educación de usuarios.
Spyware Monitorea actividad sin conocimiento del usuario. Robo de datos personales. Evitar descargas dudosas, antivirus.
Gusanos Se replican automáticamente explotando vulnerabilidades. Congestión de red, propagación rápida. Firewalls, parches, segmentación.
Keylogger Registra teclas para obtener contraseñas. Robo de credenciales bancarias. Evitar software sospechoso, 2FA.
Botnet Red de dispositivos controlados por atacante. Spam masivo, DDoS. Monitoreo de red, IDS, firewall.
SQL Injection Explotar fallos en bases de datos con código malicioso. Robo/alteración de datos. Validación de entradas, parámetros preparados.
XSS Inyección de scripts en apps web. Robo de cookies, suplantación de sesión. Sanitización, Content Security Policy.
DDoS Saturar servicio con tráfico masivo. Interrupción de servicio, pérdidas económicas. Balanceo de carga, firewalls, servicios anti-DDoS.
MitM Interceptación/modificación de comunicación. Robo de info sensible, alteración de datos. HTTPS, VPN, cifrado de datos.
